Detailed Description

template<class T>
class InfoParse::OptionHandler_< T >

Groups multiple _Option<T>-s together for some type T Can execute handling of argument string on all contained _Option<T>-s.

Used internally, not really useful outside of the internals of the project

Template Parameters
T The type for _Option<T>-s can be accepted into the group
