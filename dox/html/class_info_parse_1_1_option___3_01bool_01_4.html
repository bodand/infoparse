Detailed Description

template<>
class InfoParse::Option_< bool >

Specialized Option_ for boolean values.

Boolean values need not parse after either their option's name was present they simply dump true, otherwise false.

Stores one parsable option with a SHORT and LONG name.

LONG names are parsed like "--LONG", while SHORT names are parsed like "-SHORT", and SHORT names are one character long.
